# Around The U.S.

## Descrição
Site interativo para compartilhar fotos de lugares incríveis dos Estados Unidos.

## Funcionalidades
- Editar nome e descrição do perfil
- Adicionar novos cartões com foto e título
- Curtir cartões
- Excluir cartões
- Visualizar imagens em tamanho maior

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
